# AI-Assistant-Version-1-
It is a powerful AI-based desktop assistant designed to interact using voice commands, and control system-level operations efficiently.

The project is built using a modular architecture where features are distributed across multiple files for better performance, maintainability, and scalability.

# Project Overview
The central controller of this project is:

📂 Jarvis.py

By running this file, users can access all features of the AI Desktop Assistant.

The assistant supports both voice commands and manual text input, making it flexible and user-friendly.

# Key Features
1. Speech Recognition
- Converts speech into text
- Allows complete voice-based interaction

2. Smart Greeting System
- Greets users in multiple dynamic ways

3. System & App Control
- Open applications
- Close applications or browser tabs

4. Shut down the system (with user confirmation)

5. Smart Web Search
- Perform searches using Google
- Search and play content on YouTube
- Fully control YouTube using voice commands (play, pause, skip, etc.)

6. Emergency Feature
- Send SOS messages
- Fetch approximate user location using IP address

7. Flexible Input System
Accepts input via:
- Voice
- Manual typing

8. Basic Machine Learning / Data Operations

   It can perform basic data analysis operations on CSV files such as:
- Display rows and columns
- Sort column data
- Show summary statistics
- Basic dataset exploration

⚠️ Note: A valid .csv file link is required to perform ML/data operations.
